## Introduction

For my ICS 212 class I was assigned a semester long project called the Bank Database Application. This project displayed a menu 
that gave different options to the user, where the user could provide an account number, name, and address to create a new record and store it into the database. From there, the user could add more records, delete them, change existing records, or search for a record. 

The first iteration of the program was written in C code and later, was revised to include elements of C++. This project was a real 
challenge because C and C++ have very basic elements and don't include many features that the later Java has. It was also difficult because I had to trace pointers which can be confusing and very time consuming if not done right. 

## What I learned

This project taught me skills in tracing and helped me better understand the concept of pointers. It also strengthened my skills in problem solving and designing algorithms for programs. ICS 212 was the most time consuming class for me at the time. I actually had to take the class twice because I wasn't prepared for the time commitment the first time around. Nonetheless, that class and the Bank Database project were both valuable and helped me grow as a programmer. Now, I'm more confident in writing code and ready for what challenges I face in the future.
